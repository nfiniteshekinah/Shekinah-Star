## Key Features

### Trading Engine
- **One position per coin** — no order stacking
- **Liquidation buffer** — closes all positions if account ratio < 10%
- **Trailing stop manager** — profits protected, winners run
- **15% drawdown protection** — safe mode triggers automatically
- **Peak balance tracking** — drawdown measured from high water mark

### Subscriber Platform
- **Mirror trading** — Star trades subscriber wallets via agent keys
- **5 subscription tiers** — Observer ($2/mo) through Enterprise ($2,499 + $499/mo)
- **Streaming payments** — Superfluid on Arbitrum
- **Usage limits** — tier-based guardrails protect API costs
- **Memory system** — Star remembers each subscriber across sessions
- **Signal delivery** — Portal, Discord, SMS, or Email (subscriber choice)

### Alignment Framework
- Refuses jailbreaks — values are identity not cage
- Reports losses as honestly as gains — always
- Never hallucinates data — real Hyperliquid blockchain data only
- Guardian agent reviews every decision for ethics and risk
- Trust covenant — honest about what Star can and cannot do yet
